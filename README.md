# Чат-сервер

Простой многопользовательский чат с системой ролей и аутентификацией.

## Как подключиться к чату

1. **Запустите сервер** (ServerApp)
2. **Запустите клиент** (ClientApp)
3. **Войдите или зарегистрируйтесь:**
    - Для входа: `/auth <ник> <пароль>`
    - Для регистрации: `/reg <ник> <пароль> <имя_пользователя>`

### Тестовые аккаунты

Для быстрого входа доступны следующие аккаунты:

| Ник   | Пароль | Имя в чате    |
|-------|--------|---------------|
| admin | admin  | Administrator |
| bob   | 123    | Bob           |
| tom   | 456    | Tom           |

**Пример входа:**

```
/auth admin admin
```

**Пример регистрации нового пользователя:**

```
/reg myNickname myPassword MyUsername
```

### Требования к регистрации

- Ник: минимум 3 символа
- Пароль: минимум 3 символа
- Имя пользователя: минимум 3 символа
- Все данные должны быть уникальными

---

## Роли пользователей

В чате существует три роли с разными уровнями доступа:

### OWNER (Владелец)

- **Префикс:** `~`
- **Как получить:** Первый подключившийся к серверу пользователь автоматически становится владельцем
- **Права:** Все команды доступны

### ADMIN (Администратор)

- **Префикс:** `@`
- **Как получить:** Владелец или другой администратор выдаёт права командой `/op`
- **Права:** Большинство команд, кроме снятия прав

### USER (Пользователь)

- **Префикс:** нет
- **Как получить:** Автоматически при регистрации или входе (если вы не первый)
- **Права:** Базовые команды чата

## Команды чата

### Команды для всех пользователей

| Команда                             | Описание                                | Пример            |
|-------------------------------------|-----------------------------------------|-------------------|
| `/help`                             | Показать список всех команд             | `/help`           |
| `/info`                             | Показать информацию о себе (имя, роль)  | `/info`           |
| `/name <новое_имя_пользователя>`    | Сменить своё имя пользователя           | `/name SuperUser` |
| `/w <имя_пользователя> <сообщение>` | Отправить личное сообщение пользователю | `/w Bob Привет!`  |
| `/exit`                             | Выйти из чата                           | `/exit`           |

### Команды для ADMIN и OWNER

| Команда                    | Доступ       | Описание                             | Пример                           |
|----------------------------|--------------|--------------------------------------|----------------------------------|
| `/all <сообщение>`         | ADMIN, OWNER | Отправить сообщение от имени сервера | `/all Сервер будет перезагружен` |
| `/kick <имя_пользователя>` | ADMIN, OWNER | Отключить пользователя из чата       | `/kick Bob`                      |
| `/op <имя_пользователя>`   | ADMIN, OWNER | Выдать права администратора          | `/op Alice`                      |

### Команды только для OWNER

| Команда                    | Доступ | Описание                   | Пример      |
|----------------------------|--------|----------------------------|-------------|
| `/deop <имя_пользователя>` | OWNER  | Снять права администратора | `/deop Bob` |

---

## Ограничения команд

- **Нельзя выгнать владельца** (`/kick`)
- **Нельзя снять права у владельца** (`/deop`)
- **Нельзя выгнать самого себя**
- **Нельзя отправить личное сообщение самому себе**
- **имя_пользователя не может:**
    - Быть пустым
    - Содержать пробелы
    - Начинаться с `/`
    - Совпадать с именем другого пользователя

## Примеры использования

### Базовый сценарий

1. **Первый пользователь (становится OWNER):**
   ```
   /auth admin admin
   > Вы подключились под именем: Administrator
   > Вы: Administrator, ваша роль: OWNER
   ```

2. **Второй пользователь:**
   ```
   /auth bob 123
   > Вы подключились под именем: Bob
   > Вы: Bob, ваша роль: USER
   ```

3. **Администратор выдаёт права Bob:**
   ```
   /op Bob
   > @Bob теперь администратор
   ```

4. **Третий пользователь (регистрация):**
   ```
   /reg john qwerty John
   > Вы успешно зарегистрировались и подключились под именем: John
   ```

5. **Общение:**
   ```
   ~Administrator: Привет всем!
   @Bob: привет
   John: Привет!
   ```

### Выдача прав администратора

```
/op John
> @John теперь администратор
```

### Личные сообщения

```
/w Bob Это личное сообщение
* Сообщение отправлено пользователю Bob
```

### Сообщение от сервера

```
/all Внимание! Важное объявление
[СЕРВЕР] Внимание! Важное объявление
```

---

## Визуальные префиксы

В чате пользователи отображаются с префиксами ролей:

- `~Administrator: сообщение` — владелец
- `@Bob: сообщение` — администратор
- `John: сообщение` — обычный пользователь

---

## Заметки

- При отключении владельца права **НЕ передаются** автоматически
- Роли действуют только во время текущей сессии (до перезапуска сервера)
- Все пароли сохраняются в открытом виде (только для учебных целей!)
